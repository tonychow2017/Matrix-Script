Matrix Script Documentation

Introduction

Matrix Script is a scientific calculator with instant response. It is offered freely on iOS App Store.

Usage

Type the expression into the text box and the answer will appear (usually) immediately.

If the text box is empty, "N/A" (not available) will be shown. In case the input creates an error, "Error" will be shown.

Matrices

You can input matrices and perform matrix operations in Matrix Script.

  • To create a matrix, you may start with [ and end with ].
    Use a comma (,) as the element separator and a semi-colon(;) as the row separator.
    For example,
    >> det([1,2;3,4])
    returns -2.
  • When a matrix is passed into a scientific function, evaluation is performed entry-wise.
    For example,
    >> fact([1,2;4,5])
    returns [1,2;24,120] (factorial function).
  • Create ranges with ~.
    >> 2*(2~5)
    returns [4,6,8,10].
  • Use "eye" or "id" to create identity matrices.
    Use "one" to create matrices filled with 1 and "zero" to create matrices filled with 0.
    >> eye(2)
    returns [1,0;0,1].
    >> zero(2,3)
    returns [0,0,0;0,0,0].

Variables

You can store the calculation results into variables.
**WARNING**: This is not persistent over multiple runs (that means, the variables will be discarded upon exit). Don't store important data in this way!

  • Assign a result to a variable using the syntax:
    >> {variable_name}={expression}
  • Retrieve it using:
    >> ${variable_name}
    Note: It is important to add the dollar sign ($) before the variable name. Otherwise, we treat the name as a function name.
  • The variable $answer stores the last answer.
  • For example, the following expressions return [3,4,5,6]:
    >> a=[5,4,2,3]
    >> sort($a+1)
