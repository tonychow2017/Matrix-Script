Matrix Script Documentation

Introduction

Matrix Script is a scientific calculator with instant response. It is offered freely on iOS App Store.

Usage

Type the expression into the text box and the answer will appear (usually) immediately. If the text box is empty, "N/A" (not available) will be shown. In case the input creates an error, "Error" will be shown.

Matrices

You can input matrices and perform matrix operations in Matrix Script. To create a matrix, you may start with [ and end with ]. Use comma (,) as the element separator and (;) as the row separator. When a matrix is passed into a scientific function, evaluation is performed entry-wise. For example, det([1,2;3,4]) returns -2.

Examples

Expression Result
1+2+36
1+2*49
3^29
sin(pi/2)1
det([1,2;3,4])-2
